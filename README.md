# 全景图像拼接系统 python893

## 项目概述

本项目是基于Python和OpenCV的全景图像拼接系统，采用B/S架构，结合了HTML5和MySQL技术，致力于提供高效、稳定的全景拼接服务。

## 技术栈

- Python
- OpenCV
- MySQL
- HTML5

## 系统角色

系统设计两个角色：管理员用户和普通用户。

## 运行环境

- 系统环境：Windows
- 开发工具：Dreamweaver、Navicat for MySQL

## 功能模块

### 前台系统

- **首页**：展示系统的主要功能和入口，提供美观的用户界面。
- **图片拼接**：用户上传两张图片，系统通过OpenCV技术实现图片的全景或无缝拼接，并展示拼接结果。
- **图像管理**：用户对已拼接的图片进行查看和管理。
- **个人信息**：用户查看和修改自己的个人信息，如ID、姓名等。
- **登录/注册**：新用户进行注册，已注册用户进行登录。

### 后台管理系统

- **管理员首页**：展示管理员端的主要功能和操作入口。
- **用户信息管理**：管理员对系统内的用户信息进行增删改查。
- **图片列表管理**：管理员查看和管理所有已拼接的图片信息。
- **个人信息管理**：管理员维护自己的个人信息。
